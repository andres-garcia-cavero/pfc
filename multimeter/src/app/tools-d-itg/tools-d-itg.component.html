Log

Generate sender-side log file (default: /tmp/ITGSend.log). Generates a log file containing timing, ordering and size information about every packet sent by ITGSend. Under Windows the default log file name is "ITGSend.log".

Generate sender-side log file on a remote ITGLog instance. [logging_protocol] The first parameter sets the log server IP address (default: 127.0.0.1). The second parameter sets the transport protocol used to establish the communication between ITGSend and ITGLog: UDP or TCP (default: UDP). The name of the log file is specified by the -l option.
